# 📘 MyFinPlanner

# 🌐 Advanced Programming

이 프로젝트는 **포트폴리오 추천**, **세금 계산**, **목표 달성 시뮬레이터(그래프 포함)** 기능을 제공하는 Python 기반 재무 설계 도구입니다.

---

## 📌 프로젝트 개요

현대인의 재무 활동은 복잡하고 다양한 변수들로 구성되어 있습니다.  
하지만 대부분의 사람들은 체계적인 재무 분석이나 포트폴리오 관리 없이 투자 결정을 내리곤 합니다.

**MyFinPlanner**는 사용자가 자신의 투자 성향을 기반으로 포트폴리오를 추천받고,  
세금 계산을 통해 실제 수익을 파악하며,  
목표 달성 시뮬레이터를 통해 시간 흐름에 따른 자산 변화를 시각적으로 확인할 수 있도록 돕는 것을 목표로 합니다.

또한 그래프(PNG)를 자동 저장하여 데이터 기반 의사결정을 지원하는 간단하고 직관적인 재무 도구입니다.

---

## ✅ 주요 기능

- 📊 **포트폴리오 추천**

  - 투자 성향 설문 제공
  - 나이 기반 성향 보정
  - 추천 ETF 목록 자동 생성
  - 원형 그래프(PNG) 저장

- 💰 **세금 계산**

  - 배당소득세 자동 계산
  - 매수·매도가/수량 입력 기반 양도소득세 계산

- 🎯 **목표 시뮬레이터**

  - 목표 금액 및 기간 입력
  - 월 납입액 / 초기금 반영
  - 연 수익률 기반 성장 곡선 시뮬레이션
  - 누적 원금 vs 평가액 그래프(PNG 저장)

- 🗂️ **자동 저장 기능**

  - 모든 그래프는 `output/` 폴더에 자동 저장

- ⚙️ **설정 파일 읽기**
  - `config/config.json` 존재 시 사용자 설정 적용
  - 없을 경우 기본 설정 자동 생성
